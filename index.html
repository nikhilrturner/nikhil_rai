Projects

Quizzler App

  • Built an interactive quiz app which asks the user 15 True/False questions and maintains a score keeper to display the overall score at the end for Android as well as iOS platforms using flutter with Dart on Android Studio
  • Created a new class ‘QuizBrain’ which keeps track of all the questions, answers and the question number and incorporated this class in the main Dart file to update the question and score when the user selects an answer
  • Incorporated various Flutter compatible packages and created customisable Flutter widgets using Dart

Digital Phasemeter

  • Designed a circuit to estimate the phase difference between two sine waves with resolution 1ms
  • Used Opamp, XOR gate, AND gate and 555 timer to produce 1ms pulse for a duration equal to the phase difference
  • Employed a BCD counter to count the number of pulses and displayed it on a seven-segment display

Bluetooth Controlled Electric Car

  • Designed and built a wireless android app-controlled car capable of traversing a path filled with obstacles
  • Gained experience with RF Modules and Arduino chipsets for controlling and learnt designing on PCBs
  • Developed a data sending and receiving unit using HC-05 Bluetooth module and used light materials for the chassis for high speed acceleration of the bot to complete the designated race circuit in minimum time

Machine Learning Web App

  • Built a fully interactive Machine Learning web app with the Streamlit library in Python
  • Imported Logistic Regression, Support Vector Machines, and Random Forest Classifiers packages from scikit-learn to implement linear model, kernel method and tree based ensemble method for training the model
  • This web app allows users to choose the classification algorithm, and interactively set hyper-parameter values and displays the Accuracy, ROC curve, Confusion Matrix and Precision recall curve of our model for given dataset

Medicine Reminder

  • Built a fully functional Android app which reminds patients to take their medicine through timely alarms and makes a phone call to the emergency contact registered in the app in case of any medical emergency
  • Incorporated various media, storage and sensor components on multiple screens in the app and gained hands-on experience in developing Android applications using this block-based programming tool - MIT App Inventor

Film Club Web App
